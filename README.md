# Proyecto Final — Glow Up

Este proyecto fue desarrollado en **Java** como parte del trabajo final del semestre en **Programación Orientada a Objetos**.  
El sistema simula la gestión de una empresa ficticia llamada **Glow Up**, dirigida por la misteriosa Cabrita Sakura.  
El objetivo es aplicar los principios de la POO (herencia, composición, encapsulamiento, etc.) en un caso con varias áreas y roles diferentes.

---

##  Descripción general

El programa representa el funcionamiento interno de Glow Up, una compañía cosmética que mezcla operaciones legales y otras más “cuestionables”.  
Hay distintos tipos de usuarios dentro del sistema:

- **Cliente:** puede registrarse, iniciar sesión, comprar productos y consultar su historial.
- **Administrador de contenido:** publica, edita o elimina productos.
- **Administrador de usuarios:** gestiona cuentas y accesos.
- **Desarrollador de productos:** diseña los artículos de la marca.
- **Cabrita Sakura (Dueña):** tiene acceso completo al sistema, incluso a módulos ocultos.

La idea del proyecto es practicar el diseño de clases, relaciones y manejo de datos usando un enfoque orientado a objetos.
